Welcome to My Website!

Hello, my name is Azka, an 8th-grade middle school student learning programming from scratch. I created this website to share my experiences in exploring the world of technology and coding.

How My Learning Journey Began

I started with no prior experience at all. My interest sparked when I saw how websites and programs work, which made me curious to dive deeper into HTML, CSS, and JavaScript.

I began by learning HTML to understand how to create the basic structure of a website. After that, I moved on to CSS to style the visuals, like changing colors and layouts to make the page more appealing. I experimented with various color combinations, including my favorite pink and light blue gradient. Then, I got into JavaScript, which helped me make the website more interactive, like creating buttons that can show or hide certain sections.

Using AI to Help My Learning

Honestly, I still rely on AI to help me solve programming issues. When I face difficulties, like finding hard-to-spot bugs or when my code doesn't work as expected, AI becomes a valuable tool. With AI's help, I can fix errors and learn more quickly.

Although I'm still in the learning process and there are many things I haven't fully understood yet, each small step forward is an important part of my growth.
